# NeuralScan
## Intelligent Medical Analysis & Decision-Support System

NeuralScan is a modular, intelligent medical analysis and decision-support platform developed for **research, education, and experimental use**.  
The system integrates multiple machine learning models to analyze **breast cancer** and **diabetes** using both **clinical data** and **medical imaging**, wrapped in a professional, multi-theme analytical dashboard.
