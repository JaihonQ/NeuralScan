NeuralScan is a multi-purpose intelligent medical analysis platform designed for research and educational purposes. It integrates multiple machine learning models to analyze breast cancer and diabetes data through both clinical values and medical images.

## Key Features
- Multi-model machine learning system
- Breast cancer detection (clinical data & mammography images)
- Diabetes prediction using clinical metrics
- Smart Guide (intelligent assistant)
- User-controlled model activation and selection
- Advanced analytical dashboard with Light/Dark mode

## Machine Learning Models

### Breast Cancer
- Support Vector Machine (Mammography Images)
- Artificial Neural Network (Clinical Data)
- Random Forest (Clinical Data)

### Diabetes
- Support Vector Machine
- Artificial Neural Network

### Smart Guide
- Logistic Regression (intelligent guidance module)

## System Scale
- 45,000+ lines of Python code
- Modular ML pipelines
- 14+ analytical and control tabs

## Disclaimer
This project is intended for research and educational purposes only. Not for clinical or medical diagnosis use.
